== 简介

AsciiDoc 是一种轻量级标记语言，语法比 Markdown 更丰富，适合写长文档、技术文档和书籍。

== 特性

* 支持自动目录（`:toc:`）
* 支持章节自动编号（`:sectnums:`）
* 内建表格、引用、脚注等
* 可直接导出为 HTML、PDF、DocBook 等

== 示例表格
